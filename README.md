## 🎯 Giới thiệu

**Souvenir WebShop** là một hệ thống thương mại điện tử toàn diện được thiết kế đặc biệt cho việc kinh doanh quà lưu niệm. Ứng dụng được xây dựng trên nền tảng Spring Boot 3.2.3 với Java 21, cung cấp trải nghiệm mua sắm hiện đại và các công cụ quản lý mạnh mẽ.

Dự án này được phát triển bởi [thredeisacoder](https://github.com/thredeisacoder) và đang được phát triển tích cực với **23+ commits** và nhiều tính năng mới.


## 🚀 Bắt đầu

### 📋 Yêu cầu hệ thống

- **Java Development Kit** 21+
- **Apache Maven** 3.9+
- **Node.js** 18+ và npm 9+
- **SQL Server Express** 2019+ (hoặc SQL Server full)
- **Git** (để clone repository)

### 🔧 Cài đặt

#### 1️⃣ Clone Repository
```bash
git clone https://github.com/thredeisacoder/Souvenir-WebShop.git
cd Souvenir-WebShop
```

#### 2️⃣ Cài đặt Backend Dependencies
```bash
# Install Maven dependencies
mvn clean install

# Hoặc sử dụng Maven wrapper (không cần cài Maven)
./mvnw clean install        # Linux/Mac
./mvnw.cmd clean install    # Windows
```

#### 3️⃣ Cài đặt Frontend Dependencies
```bash
# Install SASS compiler
npm install
```

#### 4️⃣ Cấu hình Database

1. **Cài đặt SQL Server Express**:
   - Download từ [Microsoft SQL Server Express](https://www.microsoft.com/sql-server/sql-server-downloads)
   - Chọn cài đặt với Mixed Mode Authentication

2. **Tạo Database**:
```sql
-- Connect to SQL Server Management Studio hoặc sqlcmd
CREATE DATABASE SouvenirShopDB;
```

3. **Cập nhật Connection String** trong `application.properties`:
```properties
spring.datasource.username=admin
spring.datasource.password=your_password
```

#### 5️⃣ Khởi chạy ứng dụng

**Terminal 1** - Start Spring Boot Application:
```bash
mvn spring-boot:run
# Hoặc
./mvnw spring-boot:run
```

**Terminal 2** - Compile SASS (watch mode):
```bash
npm run sass
```

🎉 **Ứng dụng sẵn sàng tại**: `http://localhost:8080`

## ⚙️ Cấu hình

### 🗄️ Database Configuration

**Default `application.properties`**:
```properties
# Application
spring.application.name=sourvenir
server.port=8080

# Database Connection
spring.datasource.url=jdbc:sqlserver://localhost\\SQLEXPRESS:1433;databaseName=SouvenirShopDB;encrypt=true;trustServerCertificate=true;integratedSecurity=false
spring.datasource.driverClassName=com.microsoft.sqlserver.jdbc.SQLServerDriver
spring.datasource.username=admin
spring.datasource.password=123

# JPA/Hibernate
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.SQLServerDialect
spring.jpa.hibernate.ddl-auto=none
spring.jpa.show-sql=true
spring.jpa.properties.hibernate.format_sql=true
spring.jpa.properties.hibernate.default_schema=dbo

# Thymeleaf
spring.thymeleaf.cache=false
spring.thymeleaf.prefix=classpath:/templates/
spring.thymeleaf.suffix=.html
spring.thymeleaf.mode=HTML
spring.thymeleaf.encoding=UTF-8

# Error Handling
spring.mvc.throw-exception-if-no-handler-found=true
server.error.whitelabel.enabled=false
server.error.include-exception=true
server.error.include-stacktrace=on_param

# Logging
logging.level.project.demo=DEBUG
logging.level.org.springframework.security=DEBUG
```

### 🎨 SASS Configuration

**`package.json` SASS Script**:
```json
{
  "scripts": {
    "sass": "sass --watch src/main/resources/static/assets/scss/main.scss:src/main/resources/static/assets/css/main.css --style compressed"
  }
}
```

**Maven SASS Plugin** (pom.xml):
```xml
<plugin>
    <groupId>nl.geodienstencentrum.maven</groupId>
    <artifactId>sass-maven-plugin</artifactId>
    <version>3.7.2</version>
    <configuration>
        <sassSourceDirectory>${project.basedir}/src/main/resources/static/assets/scss</sassSourceDirectory>
        <destination>${project.basedir}/src/main/resources/static/assets/css</destination>
    </configuration>
</plugin>
```
